### CVim

Neovim configuration built on Lua **Requires Neovim v0.5.0**

### Requirements
- [ ] Git
- [ ] Neovim v0.5.0+
- [ ] Clipboard manager (ie: xsel, xclip, pbcopy)
- [ ] Python, Pip & Pynvim
- [ ] Nodejs & Neovim
